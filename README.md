# Sales Data Analysis Project

## Overview
This project analyzes sales data using SQL and Excel.  
Data was extracted from a SQLite database and analyzed using Excel Pivot Tables and charts.

## Tools Used
- SQL (SQLite)
- Excel
- Jupyter Notebook
- GitHub

## Project Steps
1. Created a sales database with customers, products, and orders
2. Performed SQL joins to combine tables
3. Exported cleaned data to Excel
4. Built Pivot Tables and charts
5. Created a Sales Performance Dashboard
